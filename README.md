# GestureLock

手势解锁视图  

1.有手势解锁图、略缩预览图  
2.都支持xib、sb实时预览  
  

如何修改样式：  
1.GestureLockView、SmallLockView .h文件中，接口提供少量样式配置  

2.AnchorView.xib：直接修改锚点样式  
GestureLockView.xib：直接修改整体布局，以及锚点视图大小  
SmallLockView.xib：修改略缩图样式
